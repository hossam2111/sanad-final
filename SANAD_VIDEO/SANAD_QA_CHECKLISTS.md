# SANAD QA Checklists — معايير القبول

## القاعدة الذهبية
"Looks good" ليست معياراً.
"5/5 L1 PASS + 3/3 L2 PASS + Human approved L3" هو المعيار.

## كيف تستخدم هذا الملف
1. لكل Asset تنتجه، انسخ الـ Checklist المناسب
2. أجب بـ PASS / FAIL على كل سؤال
3. 4 أسئلة أو أكثر FAIL = REJECTED, يعود للـ RETRY
4. 3 أسئلة أو أقل FAIL في L1 = يحتاج Human review
5. PASS كامل = APPROVED, يُنقل للـ LOCKED

## نسبة القبول
L1: 10/10 أسئلة = PASS
L2: كل الأسئلة = PASS
L3: اجتهاد بشري + gut feeling

## ملاحظات تشغيلية

### ترتيب الـ QA في سير العمل
1. Character References أولاً (قبل أي Shot)
2. Location References ثانياً (بعد اعتماد الشخصيات)
3. Shot Images ثالثاً (باستخدام References المعتمدة)
4. Video Shots رابعاً (من Images المعتمدة فقط)
5. VO وMusic بالتوازي مع الـ Shots
6. Final Assembly آخراً — بعد اعتماد كل Assets بشكل منفرد

### قاعدة الـ Retry
- أي Asset يحصل على REJECTED: يُعاد إنتاجه من الصفر
- لا يُسمح بـ "تصحيح جزئي" — الـ Asset كاملٌ أو غير موجود
- ثلاث REJECTED متتالية لنفس الـ Asset = تغيير الـ Prompt أو الـ Tool
